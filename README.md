# Finder 70.33 — Schede cliente

Mini CRM per gestire le schede cliente legate alla presentazione del Finder 70.33. Funziona da browser (anche da telefono), con dati condivisi tra tutto il team.

## Come usarlo

Apri il link del sito (Settings → Pages, dopo averlo attivato) — niente installazione, niente account richiesto.

## File

- `index.html` — l'app
- `README.md` — questo file

## Sincronizzazione dati tra colleghi

Per condividere lo stesso elenco clienti tra tutto il team serve attivare Firebase (gratuito). Vedi le istruzioni dettagliate fornite separatamente, oppure chiedi ad Arturo.
